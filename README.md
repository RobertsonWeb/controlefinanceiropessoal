# Controle Financeiro Pessoal

Contribuição de: Alexandre Zamberlan, Gilberto Orengo, Marcos Cassal, Sylvio Vieira e Robertson E.S.*

- Frontend: Framework Bootstrap 3 (Biblioteca django-bootstrap3).
